Introduction

As the craft of Web design continues to evolve, we’re recognizing the need to develop thoughtful design systems, rather than creating simple collections of web pages. — Brad Frost

This style guide is inspired by that idea of Brad Frost. The idea is not new but really helpful and rarely is applied in building websites in Viet Nam. We agree with him that it is important to have a "thoughtful design systems, rather than creating simple collections of web pages". We wanted UC designs could have a consistent style throughtout many pages of the products with less time spending repeat ourselves but more time building new good stuffs for our users. Thus we made this page, our own design system, a collection of independent and small atomic parts that can be combined to make different interfaces.

Grid

We are mostly using Bourbon Neat and Flexbox for our grid system. They are really simple, easy to use and really fit to our need. Since Neat is a really lightweight grid framework, it enabled us to use it along with using Flexbox for our smaller components.

This is a simple example of using Neat to layout a page. .wrapper { @include outer-container;} .sidebar { @include span-columns(2);} .main { @include span-columns(10);}

Colors

Colors should be used consistently on different pages. We wanted to have an unique, clear & modern theme that get along with our product's name: Conztellation - a sky full of stars. Therefore, we decided to go with dark theme with attractive colors.

Alerts

These alert boxes are designed to help users get used to the site. They should be used to display information that is short, helpful for users.

Success! This one let the user knows that they have done something successfully on the site.

Hey, This one is helpful when you want to provide more information about the site.

Warning! This one let the user knows that they should be cautious with something.

Error! This one let the user knows that they have failed in doing something on the site.
